## 🧾 Описание

**Предметная область** – кружки дошкольного развития (творческие, спортивные, познавательные).  
Приложение позволяет:

- вести список кружков (вид, название, руководитель, расписание, стоимость, кол-во учеников);
- управлять пользователями системы (только администратор);
- выполнять аналитические запросы: поиск, сортировка, фильтрация;
- разграничивать права: администратор может **редактировать**, пользователь – только **просматривать** и **искать**;

Данные хранятся в текстовых файлах (`users.txt`, `clubs.txt`) и загружаются в динамические структуры (`std::vector`).

## ⚙️ Функциональные возможности

### 👑 Администратор

| Модуль | Действия |
|--------|----------|
| **Управление учётными записями** | просмотр, добавление, редактирование, удаление пользователей |
| **Редактирование данных кружков** | CRUD операции: добавление, удаление, изменение кружков |
| **Обработка данных** | <ul><li>удаление кружков с количеством участников < N</li><li>изменение расписания кружка</li><li>поиск (по виду, названию, руководителю, диапазону цен)</li><li>сортировка (по названию, цене, ученикам, виду)</li></ul> |

### 👤 Пользователь

| Модуль | Действия |
|--------|----------|
| **Просмотр** | табличный вывод всех кружков |
| **Индивидуальное задание** | <ul><li>вывод кружков по виду</li><li>вывод кружков по руководителю</li></ul> |
| **Поиск** | по названию / диапазону цен |
| **Сортировка** | по названию / цене / количеству учеников |

## 🛠 Технологии

- **Язык**: C++17
- **Стандартная библиотека**: iostream, fstream, vector, string, algorithm, iomanip, sstream, cctype, limits
- **ОС**: Windows (поддержка UTF-8 через `SetConsoleOutputCP`)
- **Хранение данных**: текстовые файлы с разделителем `|`
