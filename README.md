# PreschoolClubsApp – Учёт кружков дошкольного развития

Консольное приложение на C++. Хранит данные о кружках, умеет искать, сортировать, редактировать (для админа). Разграничение прав: администратор и пользователь.

## Возможности

### Администратор
- Управление пользователями (добавлять, менять, удалять)
- Добавление/удаление/изменение кружков
- Удаление кружков, где меньше N учеников
- Изменение расписания кружка
- Поиск по виду / названию / руководителю / цене
- Сортировка по названию / цене / числу учеников / виду

### Пользователь
- Просмотр всех кружков
- Поиск по названию или диапазону цены
- Сортировка по названию / цене / ученикам
- Индивидуальные задания: вывести кружки по виду или по руководителю

## Технологии
- C++17
- Работа с файлами (users.txt, clubs.txt)
- Векторы, алгоритмы, строки

## Как запустить
1. Скомпилируй `PreschoolClubsApp.cpp` (например, в Visual Studio)
2. Запусти `.exe`
3. Логин `admin`, пароль `admin` – вход как администратор
